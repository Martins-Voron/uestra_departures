# ÜSTRA Departures (Home Assistant)

Custom Integration für den ÜSTRA Abfahrtsmonitor.

## Features
- Anzeige der nächsten Abfahrten
- Filter nach Verkehrsmittel
- Störungsmeldungen

## Installation
Über HACS als Custom Repository hinzufügen.

## Konfiguration
- Haltestelle (Name)
- Haltestellen-ID (z.B. 25000261)
